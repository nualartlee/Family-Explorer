Getting Started

Download & Install

If you haven't already, go to the Downloads page and get the latest version by clicking on the teal download button. Double-click on the downloaded file to start the installation and follow the prompts.

Open Family Explorer

Once you have installed Family Explorer, look on the desktop or in your programs list for the icon and click it to start.

If you have already been using Family Explorer, you can just double-click on any previously saved family file to open it directly.

Add Members

When you open Family Explorer for the first time, you will have a default member to start the family off. This member will already be selected and you will see the details in the top left window.

You will know what person is selected in a tree by a green outline around it. A selected relationship between to people will also have a green outline, and the data will show in the bottom left window.

Set the data for this person, and add additional people and relationships clicking the buttons in the top left window. You can also right-click on the person in the tree to add people from a context menu.

Add Additional Relationships Between Existing Members

You might want to record a particular type of relationship between two members that has not already been created when you added them. For example, a father and son with have a parent relationship by default, but you might want to record a very close relationship between them as a friendship.

To make this kind of record, right-click on one of the individuals to open the context menu. Click on 'Select' and from the submenu select the type of relationship you would like to add. The cursor will indicate that you are adding this relationship and will ask you to click on the reciprocal person.

Print & Save

To print the relationship tree whole, open the 'File' menu in the top left of the program window and select 'Print'. A preview window will display where you can select your printer and print. If you would like to print a section of interest, zoom in to it and select 'Print View' from the file menu.

You can save the family data anytime from the same 'File' menu. To reopen that file later on, just double-click it in Windows File Explorer to open it up in Family Explorer.
