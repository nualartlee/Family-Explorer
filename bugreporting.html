Bug Reporting & Feature Requests

Found A Problem?

This is only a small application, but it is likely full of mistakes that make it misbehave. When something goes wrong, please let me know so it can be corrected in later releases.

To contact me, please send me an email with as much detail as possible on the problem: what it is, how you make it happen, how it should ideally behave, etc...

Feature Requests

You can also let me know if you are using this program and would like to see a new feature added (e.g. the ability to search for text, change the default colors, etc.). I will do my best to develop a solution and add the feature future releases.
