Family Explorer

Record And View Family Relationships

Family Explorer is a simple tool that allows you to quickly create a visual representation of a family's relationships. It has been designed in the hope it will be useful for social workers and related staff.

Check the main features below, or go straight to the Downloads page to try it out now.

Ease Of Use

Creating the family tree is as easy as clicking a few buttons to create new members. Select any of the members by a mouse click and edit the relevant data. Click on any relationship between two individuals to view and make relevant notes on that particular relationship.

Visual Clarity

Different type of relationships are clearly represented by color coded links between the individuals.

Print & Save

Print the relationship tree whole, or zoom/pan to a section of interest. Save each tree in an individual file for easy archiving.
